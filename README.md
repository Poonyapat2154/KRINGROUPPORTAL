# KRINGROUP PORTAL

Modern company portal with login, dashboard, clients, edit/delete, departments, bulletin board, tasks, calendar, feedback, activity log, QR page, and Railway-ready setup.

## Login
Email: admin@kringroup.com  
Password: admin123

## Run locally
```bash
python -m pip install -r requirements.txt
python app.py
```
Open http://127.0.0.1:5000

## Deploy on Railway
Upload this folder to GitHub, deploy from GitHub on Railway, then generate a public domain from service Settings > Networking.
